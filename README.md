# 饥荒联机版 Mod 开发完全指南

> 从零基础到高级开发，系统性地掌握 Don't Starve Together Mod 开发

---

## 关于本教程

本教程是一份面向饥荒联机版 Mod 开发者的完整指南，涵盖了从 Lua 语言基础到游戏各系统（战斗、食物、世界生成、UI 等）的深入讲解。

**目标读者：** 对饥荒感兴趣、想要开发 Mod 但缺乏系统学习资料的玩家。

## 教程结构

本教程共分为 **六大篇**，按照学习路径循序渐进：

| 篇章 | 内容 | 章节 |
|------|------|------|
| **第一篇：基础篇** | 进入饥荒的世界 | 第 1-4 章 |
| **第二篇：核心系统篇** | 理解饥荒的"骨架" | 第 5-10 章 |
| **第三篇：视觉与交互篇** | 让游戏"活"起来 | 第 11-17 章 |
| **第四篇：游戏机制篇** | 理解饥荒的"血肉" | 第 18-28 章 |
| **第五篇：世界系统篇** | 理解饥荒的"土地" | 第 29-38 章 |
| **第六篇：进阶篇** | 成为高手 | 第 39-46 章 |

此外，附录中还提供了大量速查手册和索引，方便随时查阅。

## 如何使用

- **左侧目录栏**：点击章节标题即可跳转
- **搜索功能**：页面顶部搜索框可全文检索
- **翻页导航**：每章底部有"上一章/下一章"导航
